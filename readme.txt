Aplicativo web front-end que implementa a técnica Pomodoro com ciclos de foco e pausa, utilizando JavaScript 
Vanilla para lógica de temporização persistente baseada em timestamp, SVG para barra circular 
de progresso animada, e LocalStorage para persistência de cronograma e histórico de ciclos. O sistema permite 
cadastro dinâmico de matérias com horários, alertas sonoros via HTML5 Audio e interface responsiva baseada em CSS Grid/Flex.
